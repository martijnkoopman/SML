Detailed Description

Read and write images as PNM files (Portable Any Map).

This class supports 3 types:

Portable Bit Map (PBM) http://netpbm.sourceforge.net/doc/pbm.html This file format supports binary images (0 or 1 values).

Portable Gray Map (PGM) http://netpbm.sourceforge.net/doc/ppm.html This file format supports grayscale images. 8 and 16 bit.

Portable Pixel Map (PPM) http://netpbm.sourceforge.net/doc/ppm.html This file format supports RGB images.
